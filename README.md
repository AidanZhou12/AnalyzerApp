# Streamlit Log Analyzer

A Streamlit application for analyzing Windows Event Viewer CSV exports

## Features

- Upload Event Viewer CSV files
- Select timeframe
- Count total events
- Count errors, warnings, and informational events
- Display summary statistics
- Display a pie chart of event severity

## Requirements

- Python 3.12.3+
- streamlit
- matplotlib

## Run

'''bash
streamlit run AnalyzerApp.py
